# FocusMeter

## Трекинг фокуса за компьютером

FocusMeter — приложение для Windows, которое помогает отслеживать рабочее время, отвлечения и периоды бездействия без перегруженного интерфейса и лишних действий.

## Основные возможности

- Учет активного времени в рабочих приложениях.
- Учет отвлекающих приложений и времени вне фокуса.
- Определение бездействия по вводу с клавиатуры и мыши.
- Напоминания о перерыве и о возвращении к работе.
- Гибкая настройка правил для приложений.
- Виджет для быстрого контроля текущего состояния.
- Подробная статистика по дню и по приложениям.

## Как это работает

FocusMeter отслеживает:

- активное окно;
- наличие пользовательского ввода;
- тип приложения: рабочее, отвлекающее, исключенное или нейтральное.

На основе этого приложение относит время к одному из состояний:

- **Работа** — активное рабочее приложение и есть ввод.
- **Отдых / отвлечение** — активное отвлекающее приложение или пауза вне работы.
- **Бездействие** — долго нет ввода.
- **Нейтральное состояние** — приложение без правила не искажает рабочий таймер.

## Установка и запуск

1. Скачайте `FocusMeter.exe` из [последнего релиза](https://github.com/DenissonVoid/FocusMeter/releases/latest).
2. Поместите файл в удобную папку.
3. Запустите приложение.

При первом запуске рядом с программой будут созданы:

- `config.json` — настройки приложения;
- `focusmeter.db` — база статистики;
- `app_rules.json` — правила для приложений.

## Для кого это приложение

FocusMeter подойдет, если вы хотите:

- понимать, сколько времени реально уходит на работу;
- отделять рабочие приложения от отвлекающих;
- получать мягкие напоминания о перерывах;
- видеть простую и полезную статистику без сложных настроек.

## Приватность

FocusMeter не записывает текст с клавиатуры и не сохраняет содержимое экрана.

Приложение использует только:

- факт наличия ввода;
- имя процесса;
- заголовок активного окна;
- данные для локальной статистики.
