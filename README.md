# FocusMeter

## Трекинг фокуса за компьютером

FocusMeter помогает отслеживать рабочее время, отвлечения и периоды бездействия в течение дня.

## Актуальная версия

**Текущий релиз: `v0.3.0`**

Скачать последнюю Windows-сборку можно в [Releases](https://github.com/DenissonVoid/FocusMeter/releases/latest).

## Что нового в `v0.3.0`

- Переработан интерфейс: главный экран, статистика, каталог приложений и виджет.
- Добавлены правила приложений (рабочее / отвлекающее / исключенное / без правила).
- Улучшены стабильность трекинга и поведение окон.
- Логика уведомлений и состояния фокуса стала более предсказуемой.

## Основные возможности

- Учет активного времени в рабочих приложениях.
- Учет отвлекающих приложений и времени вне фокуса.
- Определение бездействия по вводу мыши и клавиатуры.
- Напоминания о перерыве и о возврате к фокусу.
- Каталог приложений с правилами и избранным.
- Отдельный виджет текущего состояния.
- Подробная статистика по дню и по приложениям.

## Как это работает

FocusMeter отслеживает:

- активное окно;
- факт ввода;
- тип приложения по правилам.

Состояния:

- **Работа** — активное рабочее приложение и есть ввод.
- **Отвлечение** — активное отвлекающее приложение.
- **Idle** — долго нет ввода.
- **Исключено** — приложение исключено из логики фокуса.
- **Нейтрально** — приложение без правила.

## Установка и запуск

### Windows (релиз `.exe`)

1. Скачайте `FocusMeter.exe` из [последнего релиза](https://github.com/DenissonVoid/FocusMeter/releases/latest).
2. Поместите файл в удобную папку.
3. Запустите приложение.

При первом запуске рядом с программой создаются:

- `config.json` — настройки;
- `focusmeter.db` — база статистики;
- `app_rules.json` — правила приложений.

### macOS (запуск из исходников)

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python main_gui.py
```

Для CLI-режима:

```bash
python main.py --cli
```

Для корректной работы трекинга на macOS выдайте Python-приложению права:

- `System Settings -> Privacy & Security -> Accessibility`
- `System Settings -> Privacy & Security -> Input Monitoring`
- при необходимости для заголовков окон: `Screen Recording`

## Правила приложений

- Все имена процессов нормализуются к lowercase.
- Сопоставление выполняется без учета регистра.
- Удобный способ узнать имя процесса: открыть каталог приложений в FocusMeter и назначить правило прямо из списка.

## Приватность

FocusMeter не записывает текст с клавиатуры и не сохраняет содержимое экрана.

Используются только:

- факт ввода;
- имя процесса;
- заголовок активного окна;
- локальная статистика в `focusmeter.db`.
